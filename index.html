HTML

  • The head element contains information about the webpage
  • The body element represent the visible content shown to the user
  • Elements, such as the meta and title are nested inside the head element
  • Hypertext Markup Language (HTML) is used to give the content structure
  • The following element types are commonly used to add content to pages: Text, Image Header and List elements
  • Here are some common element types that are used to divide and describe a page: Meta related, Content Division and Semantic elements
  • Declarations are not case-sensitive; HTML elements are case-sensitive

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • We can refer to a color by its hexadecimal code, or hex #
  • Assigning colors in CSS can be done in many ways, including by semantic references
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
